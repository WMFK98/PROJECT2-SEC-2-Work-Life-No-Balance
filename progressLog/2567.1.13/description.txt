หลังจาก marge และ refactor code ของเพื่อน จนปรับให้เข้ากันได้แล้วยังได้ออกแบบให้ item ของเราสามารถใช้ในรูปแบบ stack กันได้อีกด้วย ซึ่งวันนี้เราได้มาอธิบายส่วนประกอบของ  code ที่มีการปรับเปลี่ยนไปและรูปแบบของการทำงานและการเพิ่มของ item ในอนาคตให้เพื่อนเพื่อนทุกคนเข้าใจเพื่อให้เขียนมาในรูปแบบที่เป็นระบบมากขึ้น ตอนนี้ใน task3 ที่สามจึงได้ให้ทุกคนไปเปลี่ยนรูปแบบการทำงานของitem ของตนให้มีความเข้ากันได้กับโครงสร้างนี้
